# Poker Club Frontend

Фронтенд приложения для покерного клуба в стиле Liquid Glass.

## Роли

- **PLAYER (Гость)** — Турниры и расписание, Рейтинги, Профиль
- **ADMIN** — Всё выше + Настройки (серии, сезоны, блайнды, клубы, меню, TV), управление live-турнирами
- **WAITER (Официант)** — Только турниры: список игроков и приём заказов
- **TV** — Отображение live-статистики на телевизоре

## Запуск

```bash
npm install
npm run dev
```

## Переменные окружения

- `VITE_API_URL` — URL бэкенда (по умолчанию http://localhost:3000)

## TV-режим

Открыть `/tv` для отображения live-турнира. Можно передать `?tournamentId=...` или `?clubId=...` для выбора турнира/клуба.
